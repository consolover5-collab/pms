# Заезд и выезд гостей (Check-in / Check-out)

Заезд (Check-in) и выезд (Check-out) - ключевые операции Front Desk.

## Check-in (Заселение гостя)

### Требования для заселения

Перед заселением система проверяет:

1. ✅ **Статус бронирования** = Confirmed
2. ✅ **Номер назначен** (или выбран при заселении)
3. ✅ **Номер чистый** (Clean или Inspected)
4. ✅ **Номер свободен** (Vacant)
5. ✅ **Тип номера соответствует** бронированию
6. ✅ **Нет конфликтов** с другими бронированиями

### Процедура заселения

#### Вариант 1: Из списка заездов

1. Перейдите в **Bookings → Arrivals Today**
2. Найдите бронирование гостя
3. Нажмите **Check In**

#### Вариант 2: Из карточки бронирования

1. Откройте бронирование
2. Убедитесь, что номер назначен
3. Нажмите **Check In**

#### Если номер не назначен

1. Откройте бронирование
2. Нажмите **Edit**
3. Выберите доступный номер из списка
4. Сохраните
5. Нажмите **Check In**

### Что происходит при Check-in

| Элемент | Изменение |
|---------|-----------|
| Booking status | `confirmed` → `checked_in` |
| Booking actualCheckIn | Записывается текущее время |
| Room occupancyStatus | `vacant` → `occupied` |

### Ошибки при Check-in

| Сообщение | Причина | Решение |
|-----------|---------|---------|
| "Room must be assigned" | Номер не назначен | Назначьте номер через Edit |
| "Room is not ready" | Номер грязный | Пометьте номер чистым |
| "Room is not vacant" | Номер занят | Выберите другой номер |
| "Room type does not match" | Тип номера не совпадает | Измените тип в бронировании |
| "Room already has a checked-in guest" | Конфликт | Проверьте другие бронирования |

## Check-out (Выселение гостя)

### Требования для выселения

- ✅ **Статус бронирования** = Checked In

### Процедура выселения

#### Вариант 1: Из списка выездов

1. Перейдите в **Bookings → Departures Today**
2. Найдите бронирование
3. Нажмите **Check Out**

#### Вариант 2: Из карточки бронирования

1. Откройте бронирование со статусом **Checked In**
2. Нажмите **Check Out**

### Что происходит при Check-out

| Элемент | Изменение |
|---------|-----------|
| Booking status | `checked_in` → `checked_out` |
| Booking actualCheckOut | Записывается текущее время |
| Room occupancyStatus | `occupied` → `vacant` |
| Room housekeepingStatus | *текущий* → `dirty` |

> **Важно:** После выселения номер автоматически становится **грязным** и требует уборки.

## Типичные сценарии

### Ранний заезд (Early Check-in)

Гость приехал раньше стандартного времени заезда:
1. Убедитесь, что номер готов (чистый и свободный)
2. Выполните стандартный Check-in
3. При необходимости добавьте примечание в бронирование

### Поздний выезд (Late Check-out)

Гость просит задержаться:
1. Проверьте, нет ли заезда на этот номер сегодня
2. Если номер свободен - согласуйте с гостем
3. Выполните Check-out когда гость уедет

### Неявка (No Show)

Гость не приехал в день заезда:
1. В конце дня бронирование можно пометить как **No Show**
2. Из статуса No Show можно отменить бронирование

### Смена номера (Room Move)

Гость просит переселить в другой номер:
1. Откройте бронирование (статус Checked In)
2. Нажмите **Edit**
3. Выберите новый номер
4. Сохраните
5. Старый номер станет грязным (dirty)

### Утро (начало смены)
- [ ] Проверить Dashboard - сколько заездов ожидается
- [ ] Проверить, все ли номера для заездов чистые
- [ ] Подготовить регистрационные карты (если есть)

### Вечер (конец смены)
- [ ] Проверить, все ли ожидаемые гости заселены
- [ ] Проверить No Show - пометить неявившихся
- [ ] Передать смену - кто ещё должен заехать/выехать
