# Конфигурация (Configuration)

Модуль конфигурации позволяет настраивать справочники и параметры системы.

## Расположение
- URL: `/configuration`

## Разделы конфигурации

### 1. Room Types (Типы номеров)

Управление категориями номеров.

**URL:** `/configuration/room-types`

#### Поля

| Поле | Описание | Обязательное |
|------|----------|--------------|
| **Code** | Короткий код (STD, SUP, DLX) | ✅ Да |
| **Name** | Полное название | ✅ Да |
| **Max Occupancy** | Максимальная вместимость | ✅ Да |
| **Base Rate** | Базовая цена за ночь | ✅ Да |
| **Description** | Описание | Нет |
| **Sort Order** | Порядок сортировки | Нет |

#### Операции

- **Создать**: `/configuration/room-types/new`
- **Редактировать**: `/configuration/room-types/[id]/edit`
- **Удалить**: Кнопка Delete в списке

> **Важно:** Нельзя удалить тип номера, если есть связанные номера или бронирования.

#### Примеры типов номеров

| Code | Name | Max Occ. | Base Rate |
|------|------|----------|-----------|
| STD | Standard | 2 | 4500 |
| SUP | Superior | 3 | 5500 |
| DLX | Deluxe | 4 | 7000 |
| STE | Suite | 4 | 10000 |

---

### 2. Rate Plans (Тарифные планы)

Управление тарифами и ценовыми стратегиями.

**URL:** `/configuration/rate-plans`

#### Поля

| Поле | Описание | Обязательное |
|------|----------|--------------|
| **Code** | Короткий код (RACK, CORP, PROMO) | ✅ Да |
| **Name** | Название тарифа | ✅ Да |
| **Base Rate** | Базовая цена (если фиксированная) | Нет |
| **Description** | Описание условий | Нет |
| **Is Active** | Активен ли тариф | Да (по умолчанию) |

#### Операции

- **Создать**: `/configuration/rate-plans/new`
- **Редактировать**: `/configuration/rate-plans/[id]/edit`
- **Удалить**: Кнопка Delete в списке

#### Примеры тарифов

| Code | Name | Description |
|------|------|-------------|
| RACK | Rack Rate | Стандартный публичный тариф |
| CORP | Corporate | Корпоративная скидка |
| PROMO | Promotional | Акционный тариф |
| WKND | Weekend | Тариф выходного дня |

---

### 3. Property Settings (Настройки отеля)

Общие настройки отеля.

**URL:** `/configuration/property`

#### Основная информация

| Поле | Описание |
|------|----------|
| **Property Name** | Название отеля |
| **Code** | Короткий код отеля |
| **Address** | Адрес |
| **City** | Город |
| **Country** | Страна (код ISO) |

#### Операционные настройки

| Поле | Описание | Пример |
|------|----------|--------|
| **Check-in Time** | Стандартное время заезда | 14:00 |
| **Check-out Time** | Стандартное время выезда | 12:00 |
| **Timezone** | Часовой пояс | Europe/Moscow |
| **Currency** | Валюта | RUB |

#### Ёмкость

| Поле | Описание |
|------|----------|
| **Number of Rooms** | Общее количество номеров |
| **Number of Floors** | Количество этажей |

---

## Порядок настройки

При первоначальной настройке системы рекомендуется:

1. **Property Settings** - основная информация об отеле
2. **Room Types** - категории номеров
3. **Rate Plans** - тарифы
4. Затем создать номера и начать работу

## Изменение конфигурации

### Добавление нового типа номера

1. Перейдите в **Configuration → Room Types**
2. Нажмите **Add Room Type**
3. Заполните код, название, вместимость, цену
4. Сохраните
5. Создайте номера этого типа в базе данных

### Создание нового тарифа

1. Перейдите в **Configuration → Rate Plans**
2. Нажмите **Add Rate Plan**
3. Заполните код и название
4. Укажите базовую цену (если фиксированная)
5. Установите флаг Active
6. Сохраните

### Изменение времени заезда/выезда

1. Перейдите в **Configuration → Property Settings**
2. Измените **Check-in Time** или **Check-out Time**
3. Нажмите **Save Settings**

## Влияние на систему

| Изменение | Влияние |
|-----------|---------|
| Новый Room Type | Можно создавать номера и бронирования этого типа |
| Удаление Room Type | Существующие бронирования не затрагиваются |
| Деактивация Rate Plan | Нельзя выбрать при новых бронированиях |
| Изменение Check-in Time | Информационное, не влияет на бизнес-логику |
