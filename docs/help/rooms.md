# Управление номерами (Rooms)

Модуль управления номерами позволяет контролировать состояние номерного фонда и координировать работу хозяйственной службы.

## Расположение
- URL: `/rooms`
- Детали номера: `/rooms/[id]`

## Обзор номеров (Room Grid)

Страница показывает все номера в виде сетки с визуальными индикаторами состояния.

### Информация на карточке номера

- **Номер** (например, 301)
- **Тип номера** (STD, SUP, PRM)
- **Этаж**
- **Статус уборки** (цветовая метка)
- **Статус занятости** (Vacant/Occupied)

### Цветовая кодировка

| Статус | Цвет | Описание |
|--------|------|----------|
| Clean + Vacant | 🟢 Зелёный | Готов к заселению |
| Dirty | 🔴 Красный | Требует уборки |
| Occupied | 🔵 Синий | Занят гостем |
| Inspected | 🔵 Синий (тёмный) | Проверен |
| Out of Order | ⚫ Чёрный | Не работает |

## Статусы Housekeeping

### Clean (Чистый)
- Номер убран и готов к заселению
- Можно заселять гостей

### Dirty (Грязный)
- Номер требует уборки
- Нельзя заселять гостей
- Автоматически устанавливается после check-out

### Inspected (Проверен)
- Номер убран и проверен супервайзером
- Высший уровень готовности
- Можно заселять гостей

### Pickup (Лёгкая уборка)
- Требует минимальной уборки
- Используется когда гость был в номере очень короткое время

### Out of Order (Не работает)
- Номер снят с продажи
- Не учитывается в доступности
- Используется при серьёзных проблемах (ремонт, поломка)

### Out of Service (На обслуживании)
- Номер временно недоступен
- Остаётся в инвентаре (можно продать если срочно нужно)
- Используется при мелком обслуживании

## Действия с номером

### Изменение статуса уборки

1. Перейдите в **Rooms**
2. Кликните на номер
3. В разделе **Quick Actions** выберите действие:
   - **Mark Clean** - пометить чистым
   - **Mark Dirty** - пометить грязным
   - **Mark Inspected** - пометить проверенным
   - **Out of Order** - вывести из строя
   - **Return to Service** - вернуть в работу

### Логика переходов

```
Dirty → Clean → Inspected
  ↑       ↓        ↓
  └───────┴────────┘
         (Mark Dirty)

Любой → Out of Order → Dirty (Return to Service)
```

## Фильтрация

На странице номеров можно фильтровать по:
- **Housekeeping Status**: Clean, Dirty, Inspected, Out of Order
- **Occupancy**: Vacant, Occupied

## Статистика

В верхней части страницы отображается:
- Всего номеров
- Свободных (Vacant)
- Занятых (Occupied)
- Чистых (Clean)
- Грязных (Dirty)

## Рабочий процесс Housekeeping

### Утро

1. Откройте **Rooms** с фильтром **Dirty**
2. Это список номеров для уборки
3. По мере уборки помечайте номера **Clean**

### После выездов

- Система автоматически помечает номера **Dirty** при check-out
- Проверяйте список грязных номеров

### Проверка супервайзером

1. Откройте **Rooms** с фильтром **Clean**
2. Проверьте номер физически
3. Пометьте **Inspected**

### Проблемы с номером

Если обнаружена проблема:
1. Откройте номер
2. Нажмите **Out of Order**
3. Номер будет недоступен для бронирования
4. После устранения проблемы: **Return to Service** → **Mark Clean**
