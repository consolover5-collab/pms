## Основные понятия

### Статусы бронирования

| Статус | Описание | Действия |
|--------|----------|----------|
| **Confirmed** | Подтверждённое бронирование | Можно редактировать, отменять, заселять |
| **Checked In** | Гость заселён | Можно выселять |
| **Checked Out** | Гость выселен | Только просмотр |
| **Cancelled** | Отменено | Только просмотр |
| **No Show** | Неявка | Можно отменить |

### Статусы номеров

**Housekeeping (уборка):**
- **Clean** - Чистый, готов к заселению
- **Dirty** - Требует уборки
- **Inspected** - Проверен супервайзером
- **Pickup** - Требует лёгкой уборки
- **Out of Order** - Не работает (снят с продажи)
- **Out of Service** - На обслуживании (можно продать)

**Occupancy (занятость):**
- **Vacant** - Свободен
- **Occupied** - Занят

## Рабочие процессы

### Начало смены
1. Откройте Dashboard (главную страницу)
2. Проверьте количество заездов на сегодня
3. Проверьте количество выездов на сегодня
4. Проверьте статус номеров (сколько грязных)

### Заезд гостя (Check-in)
1. Перейдите в **Bookings → Arrivals Today**
2. Найдите бронирование гостя
3. Убедитесь, что номер назначен и чист
4. Нажмите **Check In**

### Выезд гостя (Check-out)
1. Перейдите в **Bookings → Departures Today**
2. Найдите бронирование гостя
3. Нажмите **Check Out**
4. Номер автоматически станет "Dirty"

### Уборка номера
1. Перейдите в **Rooms**
2. Найдите грязный номер (красный статус)
3. Нажмите на номер
4. Нажмите **Mark Clean**
