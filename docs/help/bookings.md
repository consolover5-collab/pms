# Бронирования (Bookings)

Управление бронированиями - основной модуль системы. Здесь вы создаёте, просматриваете и управляете всеми резервациями.

## Расположение
- URL: `/bookings`

## Вкладки (Views)

### All (Все бронирования)
Полный список бронирований. Можно фильтровать по статусу:
- Confirmed
- Checked In
- Checked Out
- Cancelled
- No Show

### Arrivals Today (Заезды сегодня)
- URL: `/bookings?view=arrivals`
- Показывает бронирования с датой заезда = сегодня
- Статусы: Confirmed (ожидают) и Checked In (уже заселены)
- Быстрое действие: **Check In**

### Departures Today (Выезды сегодня)
- URL: `/bookings?view=departures`
- Показывает бронирования с датой выезда = сегодня
- Статусы: Checked In (ожидают выезда) и Checked Out (уже выехали)
- Быстрое действие: **Check Out**

### In-House (Гости в отеле)
- URL: `/bookings?view=inhouse`
- Показывает только бронирования со статусом **Checked In**
- Полезно для просмотра текущей загрузки

## Создание бронирования

1. Нажмите **+ New Booking**
2. Заполните обязательные поля:
   - **Guest** - выберите существующего гостя или создайте нового
   - **Room Type** - категория номера
   - **Check-in Date** - дата заезда
   - **Check-out Date** - дата выезда
   - **Adults** - количество взрослых
3. Опциональные поля:
   - **Room** - конкретный номер (можно назначить позже)
   - **Rate Plan** - тарифный план
   - **Children** - количество детей
   - **Notes** - примечания
4. Нажмите **Create Booking**

> **Совет:** Номер можно не назначать сразу. Это можно сделать при редактировании или при заселении.

## Просмотр бронирования

Кликните на **Confirmation Number** (номер подтверждения) для просмотра деталей:

- Информация о госте (имя, email, телефон)
- Даты проживания
- Назначенный номер и тип
- Тарифный план
- Сумма
- Статус
- Фактическое время заезда/выезда

## Редактирование бронирования

1. Откройте бронирование
2. Нажмите **Edit**

**Ограничения по статусу:**
- **Confirmed** - можно редактировать всё
- **Checked In** - можно изменить только примечания и номер
- **Checked Out / Cancelled** - редактирование недоступно

## Статусы и переходы

```
Confirmed → Checked In → Checked Out
    ↓
Cancelled ← (только из Confirmed или No Show)
```

### Правила переходов:

| Из статуса | В статус | Условие |
|------------|----------|---------|
| Confirmed | Checked In | Номер назначен, чистый, свободный |
| Checked In | Checked Out | - |
| Confirmed | Cancelled | - |
| No Show | Cancelled | - |
| Checked In | Cancelled | ❌ Невозможно (сначала check-out) |

## Поиск бронирования

В таблице отображается:
- Номер подтверждения (кликабельный)
- Имя гостя
- Номер комнаты
- Даты заезда/выезда
- Статус
- Сумма

Используйте фильтры по статусу для быстрого поиска.

## Частые операции

### Назначить номер
1. Откройте бронирование
2. Нажмите **Edit**
3. Выберите **Room** из списка
4. Сохраните

### Отменить бронирование
1. Откройте бронирование со статусом **Confirmed**
2. Нажмите **Cancel Booking**
3. Подтвердите отмену

### Изменить даты
1. Откройте бронирование
2. Нажмите **Edit**
3. Измените даты
4. Сохраните

> **Важно:** Нельзя изменить даты для бронирований со статусом Checked In, Checked Out или Cancelled
