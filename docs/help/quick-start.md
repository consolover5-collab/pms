# Quick Start - Быстрый старт

## Первый вход в систему

1. Откройте браузер и перейдите по адресу: `http://localhost:3000`
2. Вы увидите **Dashboard** с информацией о текущем состоянии отеля

## Основные действия

### Создать бронирование

1. Нажмите **Bookings** на главной странице
2. Нажмите **+ New Booking**
3. Заполните форму:
   - Выберите гостя (или создайте нового)
   - Выберите тип номера
   - Укажите даты заезда и выезда
   - Укажите количество гостей
4. Нажмите **Create Booking**

### Заселить гостя

1. Перейдите в **Bookings**
2. Выберите вкладку **Arrivals Today**
3. Найдите бронирование
4. Кликните на номер подтверждения
5. Назначьте номер (если не назначен)
6. Нажмите **Check In**

> **Важно:** Номер должен быть чистым (Clean) и свободным (Vacant)

### Выселить гостя

1. Перейдите в **Bookings → Departures Today**
2. Найдите бронирование
3. Нажмите **Check Out**

### Пометить номер чистым

1. Перейдите в **Rooms**
2. Найдите грязный номер (красная метка)
3. Кликните на номер
4. Нажмите **Mark Clean**

## Навигация

| Раздел | URL | Описание |
|--------|-----|----------|
| Dashboard | `/` | Главный экран со статистикой |
| Бронирования | `/bookings` | Список всех бронирований |
| Заезды | `/bookings?view=arrivals` | Заезды на сегодня |
| Выезды | `/bookings?view=departures` | Выезды на сегодня |
| In-House | `/bookings?view=inhouse` | Гости в отеле |
| Номера | `/rooms` | Управление номерами |
| Гости | `/guests` | Профили гостей |
| Настройки | `/configuration` | Конфигурация системы |

## Цветовая индикация

### Статусы бронирований
- 🔵 **Синий** - Confirmed (подтверждено)
- 🟢 **Зелёный** - Checked In (заселён)
- ⚪ **Серый** - Checked Out (выселен)
- 🔴 **Красный** - Cancelled (отменено)
- 🟡 **Жёлтый** - No Show (неявка)

### Статусы номеров
- 🟢 **Зелёный** - Clean (чистый)
- 🔴 **Красный** - Dirty (грязный)
- 🔵 **Синий** - Inspected (проверен) / Occupied (занят)
- ⚫ **Тёмный** - Out of Order (не работает)
