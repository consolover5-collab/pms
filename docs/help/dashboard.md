# Dashboard - Главный экран

Dashboard предоставляет обзор текущего состояния отеля, аналогично экрану "House Status" в Oracle Opera (Shift+F3).

## Расположение
- URL: `/` (главная страница)

## Разделы Dashboard

### 1. Room Summary (Сводка по номерам)

Показывает текущее состояние номерного фонда:

| Показатель | Описание |
|------------|----------|
| **Total Rooms** | Общее количество номеров в отеле |
| **Occupied** | Количество занятых номеров |
| **Vacant** | Количество свободных номеров |
| **Occupancy %** | Процент загрузки отеля |

### 2. Today's Activity (Активность на сегодня)

Показывает движение гостей:

| Показатель | Описание | Ссылка |
|------------|----------|--------|
| **Arrivals** | Заезды сегодня (ожидающие / всего) | `/bookings?view=arrivals` |
| **Departures** | Выезды сегодня (ожидающие / всего) | `/bookings?view=departures` |
| **In-House** | Гости в отеле сейчас | `/bookings?view=inhouse` |

> **Совет:** Цифра перед слешем показывает количество ожидающих действия (не заселённых / не выселенных)

### 3. Housekeeping (Хозяйственная служба)

Показывает статус уборки номеров:

| Статус | Цвет | Описание |
|--------|------|----------|
| **Clean** | 🟢 Зелёный | Чистые, готовы к заселению |
| **Dirty** | 🔴 Красный | Требуют уборки |
| **Inspected** | 🔵 Синий | Проверены супервайзером |
| **Out of Order/Service** | ⚫ Серый | Не доступны |

### 4. Pending Arrivals (Ожидающие заезда)

Список гостей, которые должны заехать сегодня, но ещё не заселены:
- Имя гостя
- Назначенный номер (или "No room assigned")
- Клик открывает детали бронирования

## Использование Dashboard

### Начало рабочего дня

1. Проверьте **Arrivals** - сколько гостей ожидается
2. Проверьте **Dirty rooms** - есть ли номера для уборки
3. Убедитесь, что для всех заездов назначены чистые номера

### В течение дня

- Следите за **Pending Arrivals** - это гости, которых ещё нужно заселить
- Следите за **Dirty count** - как продвигается уборка

### Конец дня

1. Проверьте **Departures** - все ли выехали
2. Проверьте **In-House** - соответствует ли реальной загрузке
3. Проверьте **Dirty** - все ли номера убраны для завтрашних заездов

## Быстрые действия

С Dashboard можно быстро перейти к:
- Списку бронирований (кнопка **Bookings**)
- Управлению номерами (кнопка **Rooms**)
- Профилям гостей (кнопка **Guests**)
- Настройкам (кнопка **Configuration**)
